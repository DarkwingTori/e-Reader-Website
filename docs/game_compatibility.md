# e-Reader Compatible Games

---

## Game Boy Advance

### US & Australia

| Game | What e-Reader Does |
|---|---|
| **Pokémon Ruby** | Battle special trainers, receive berries, Berry glitch fix (JP promo cards) |
| **Pokémon Sapphire** | Battle special trainers, receive berries, Berry glitch fix (JP promo cards) |
| **Super Mario Advance 4: Super Mario Bros. 3** | Unlock new levels (Level Cards), add power-ups (Power-Up Cards), watch demo runs (Demo Cards) |
| **Mario Party-e** *(card game, not cartridge)* | Scan minigame cards to play portable board game minigames |
| **Mario vs. Donkey Kong** | Hidden e-Reader support — unlocks bonus levels (US version reads JP cards partially but cannot unlock levels) |
| **Pokémon FireRed** *(JP only — removed from US)* | e-Reader functionality present in JP version, removed from US/EU releases |
| **Pokémon LeafGreen** *(JP only — removed from US)* | Same as FireRed |
| **Pokémon Emerald** *(JP only — removed from US)* | Same as FireRed |

### Japan Exclusive (e-Reader+ required)

| Game | What e-Reader Does |
|---|---|
| **Pokémon Ruby / Sapphire** | All US features + additional JP-exclusive trainer cards |
| **Pokémon Pinball: Ruby & Sapphire** | Modify gameplay — start with 9 balls, Master Ball, 99 coins; change encounter rates |
| **Mega Man Battle Network 4 (Rockman EXE 4)** | Scan mod cards for perks and bonuses |
| **Mega Man Battle Network 5 (Rockman EXE 5)** | Mod cards for abilities and items |
| **Mega Man Battle Network 6 (Rockman EXE 6)** | Mod cards — Battle Chips, Sub Chips, BugFrags, Zenny, Navi Customizer Programs |

| **F-Zero Climax** | Scan cards to unlock additional machines, race courses, and developer staff ghosts |

## GameCube

> Requires GBA with e-Reader connected via Game Boy Advance Cable to GameCube. The GameCube game must be running while the e-Reader is used on the GBA side.

### US & Australia

| Game | What e-Reader Does |
|---|---|
| **Animal Crossing** | Scan Character Cards to receive items via post office; scan Town Tune Cards to set village tune; scan Design Cards at Able Sisters; scan Classic Game Cards (Ice Climber, Mario Bros.) to receive items; scan NES game cards |
| **Pokémon Channel** | Scan 3 promo cards — "The Pikachu Star", "The Kyogre Constellation" (US) / "Jirachi" (AU), "Poké A la Card" — unlocks Smeargle paint line art |

---

### Japan Exclusive (e-Reader+ required)

| Game | What e-Reader Does |
|---|---|
| **Doubutsu no Mori+** | Scan Character Cards for secret codes redeemable in-game for items (no direct GC connectivity on original JP e-Reader — codes entered manually) |
| **Doubutsu no Mori e+** | Full Animal Crossing-equivalent e-Reader support — scan Character Cards to have villagers move in, scan Town Tune Cards to unlock new K.K. Slider songs, all features from Animal Crossing + new villagers/islanders exclusively available via cards |
| **Pikmin 2** | Scan Pikmin Puzzle Cards (6 packs) to play exclusive Pikmin-based minigames via Game Boy Player — region locked, requires JP GameCube + JP e-Reader+ + JP Pikmin 2 |

## e-Reader Functionality Removed from Western Releases

| Game | Notes |
|---|---|
| **Pokémon FireRed (US/EU)** | e-Reader menu and card support present in JP version, fully removed from localized versions |
| **Pokémon LeafGreen (US/EU)** | Same as FireRed |
| **Pokémon Emerald (US/EU)** | Same as FireRed |
| **Mega Man Battle Network 6 (US/EU)** | All e-Reader content removed — inaccessible even via cheat devices in EU/US versions |
| **Mario vs. Donkey Kong (US)** | Hidden e-Reader menu accessible via JP e-Reader+ but level cards cannot actually unlock anything on the US cartridge |
